# Supply Chain Management SQL Project

## 📌 Project Overview
This project is a Supply Chain Management database built using SQL.
It includes tables for customers, suppliers, products, orders, and order items.
The project is designed to practice SQL concepts from basic queries to advanced business analysis.

## 🗂️ Database Schema
- Customer
- Supplier
- Product
- Orders
- OrderItem

## 🛠️ SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOINS
- Window Functions
- CTE
- Subqueries
- Indexes
- Primary Keys

## 📊 Practice Questions Covered
- Customer demographics
- Revenue analysis
- Top suppliers
- Product demand ranking
- Bulk buyers
- Premium suppliers

## 🚀 Key Insights
- Identified top customers by spending
- Ranked high-demand products
- Found supplier contribution by revenue
- Analyzed yearly order trends

## 💻 Tools Used
- MySQL
- SQL Server
- GitHub
